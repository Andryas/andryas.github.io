What is the oldest property in Montreal?

May 29, 2022 in real-estates

Before start is essential to say that what we are exploring here is data crawled from the Centris, so it is a property sample from Montreal; my inferences here will be estimation, approximation for the true values; it is good to give an idea about the places in Montreal.

Let’s start seeing a summary of the whole data.

Table 1: Summary year built
min q25 avg med q75 max sd total_obs total_na pct_na
1690 1952 1979.7681 1992 2016 2022 40.6586 9404 1755 0.1866

The table above shows that 50% of the properties we collected were built before 1992 (med=1992). The oldest house we found is dated 1690! I don’t know if it is correct, but if it is, congratulation to Montreal; you’ve been pretty strict in dating properties.

As we can see from the table above, the oldest property we have in our database comes from Ville-Marie (1690); besides, it is the neighbourhood with the most significant number of observations, 2358.

A good visualition for this kind situation is the boxplot.

We can see the year-built distribution by each neighbourhood.

A visualization of all properties that we collected from centris. We notice that it is predominantly from 1900 later, but let’s break the year into classes.

In the plot below, we can see better the patterns; first, the oldest properties in Montreal are in the old port, so the old port lives up to the name. Second, we have many properties between [1800-1900], and I can’t stop wondering how they could have kept this information all these years. And third, the most common building age in our database from centris is between [1900-2000], followed by [2000-2022].
